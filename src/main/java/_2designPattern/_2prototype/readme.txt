
OA系统中，有一个员工提交周报的功能。每一个周五，员工都需要总结本周的工作内容，以及写出下周个工作计划

问题是，每一周的周报，内容基本大同小异，出现特殊情况的几率很小。

周报到表单看起来这个样子：

提交人：
总结：
下周计划
特殊情况
建议
意见
提交

问题是：就算其它项目不变，每周在填写周报的时候，还要填写一遍，这样的工作效率很低

对员工心情也不好，老做重复工作

解决方案：
    1.让员工可以保存周报模版，最多保存三个
    2.以后每次填写周报的时候，都可以加载以前保存好的模版，一旦加载模版，表单中就会立刻回显出模版的内容
